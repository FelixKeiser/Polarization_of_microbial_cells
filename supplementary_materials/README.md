# Supplementary Materials

This directory contains supplementary materials for the scientific paper on polarization of microbial cells.

## Directory Structure

```
supplementary_materials/
├── data/           # Raw and processed data files
├── code/           # Analysis scripts and source code
├── figures/        # Supplementary figures
├── tables/         # Supplementary tables
└── results/        # Analysis results and outputs
```

## Contents

### Data
The `data/` directory contains:
- Raw experimental data
- Processed datasets
- Metadata and data dictionaries

### Code
The `code/` directory contains:
- Analysis scripts
- Data processing pipelines
- Visualization code
- Custom functions and utilities

### Figures
The `figures/` directory contains:
- Supplementary figures not included in the main manuscript
- High-resolution versions of figures
- Raw figure files

### Tables
The `tables/` directory contains:
- Supplementary tables
- Extended datasets in tabular format

### Results
The `results/` directory contains:
- Statistical analysis outputs
- Model results
- Summary statistics

## Usage

To use these supplementary materials:

1. Extract the contents of this directory
2. Navigate to the relevant subdirectory
3. Follow the instructions in individual README files or comments in code files

## Data Formats

Common file formats used:
- `.csv` - Comma-separated values for tabular data
- `.txt` - Plain text files
- `.png`, `.pdf` - Image files
- `.py`, `.R`, `.m` - Code scripts
- `.xlsx` - Excel spreadsheets
