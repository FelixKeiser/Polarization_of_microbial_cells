# Polarization of Microbial Cells

This repository contains supplementary materials for the scientific paper on polarization of microbial cells.

## Contents

- **[Supplementary Materials](supplementary_materials/)** - Data, code, figures, tables, and results supporting the publication

## Supplementary Materials

The `supplementary_materials/` directory contains:
- **Data**: Raw and processed experimental data
- **Code**: Analysis scripts and source code
- **Figures**: Supplementary figures
- **Tables**: Supplementary tables
- **Results**: Analysis outputs

See the [supplementary materials README](supplementary_materials/README.md) for detailed information.

## Usage

To access the supplementary materials:
1. Navigate to the `supplementary_materials/` directory
2. Review the README for details on the contents and structure
3. Extract any compressed archives as needed
