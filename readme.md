**Data**

contains data created during the thesis

Data\_Step\_1+2 is a pandas .pkl file containing all simulation results and parameters for models without a brush layer

Data\_Brushlayer contains all data files for full simulations (including brush layer)

Data\_Brushlayer\_Dictionary contains the parameters for all data files in Data\_Brushlayer (only parameters deviating from the default values are given)

**Comsol**

contains the Comsol models created in the thesis

**Python**

contains python files used for reading and interpreting data

An example how this can be done is given in Example\_Usage.py
